## Wstęp

Projekt ma na celu testować, prezentować i zachować wiedzę z zakresu symfony.

## Użyte technologie:

- PHP >= 8.1
- composer
- Symfony CLI
- Symfony 6.3

### Generowanie szkieletu symfony

- w konsoli użyto komendy Symfony CLI

```bash
 symfony new new_symfony --version="6.3.*" --webapp
```
